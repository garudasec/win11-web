# win11-web 🪟

A Windows 11 experience built with pure HTML, CSS, and JavaScript — runs entirely in the browser.

---

## 🔴 Live Demo
> Open `index.html` in any browser — no setup needed!

---

## ✨ Features

- 🎯 **Smart Taskbar** — only the Windows icon opens the Start Menu (just like the real thing)
- 📋 **Start Menu** — scrollable app list with a fixed user/power bar at the bottom
- 🎥 **Animated Background** — looping video wallpaper
- ❌ **Click outside to close** — Start Menu dismisses when clicking the desktop
- ⌨️ **Keyboard support** — press `Escape` to close the Start Menu
- 📱 **Responsive** — works across different screen sizes

---

## 🚀 Getting Started

1. Clone the repo
```bash
git clone https://github.com/yourusername/win11-web.git
```

2. Open the folder and just run `index.html` in your browser — that's it!

